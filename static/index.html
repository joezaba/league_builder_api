league_builder_api

league_builder_api is a sports management Rest API for sports based organizations.

Getting Started

This project was built on the following tecnologies. The versions are not necessarily minimum prerequisits.

  • MySQL - v8.0.22
  • TypeScript - v4.1.3
  • npm - v6.14.8
  • Node.js - v12.19.0
  • Express - v4.17.1

Installation

After installing Node.js and npm: Install Typescript:

npm i -g typescript

Install Node dependencies:

npm install

NPM Scripts

Compile TypeScript:

npm run build # tsc --watch

Run Application

npm run start # node .
